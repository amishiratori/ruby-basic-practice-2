ファイル名： `fibonacci.rb`

出力するプログラムを作ってみましょう！

フィボナッチ数列は，  
下記の数列のように今の項と前項の和が次の項となるような数列です。

```

```

## 3. しりとり
ファイル名： `shiritori.rb`  

いくつかの言葉が配列に入っています．  
配列の要素がしりとりとして成り立つ順番になっているか判定するプログラムを作りましょう！  

配列が以下のように定義されたとき，
```

```

出力例は  
```
False
```

配列が以下のように定義されたとき，
```
words = ['しりとり', 'りす', 'すいか', 'かまきり', 'りんご', 'ごりら', 'らっこ', 'こもどおおとかげ', 'げんごろう', 'うんどうかい', 'いっすんぼうし', 'しすう', 'うま', 'まんどれいく']
```

出力例は
```

```

## 13. 通貨換算
ファイル名： `exchange.rb`  

変数`money`に換算したい金額（円）を，  
変数`rate`に1ドル何円かを  
整数で代入します．  

`money`に代入された金額が，  
何ドル何セントになるか，計算し，出力するプログラムを作りましょう！  

ただし，  
100セントで1ドルであり，  
1セント未満の端数は切り捨ててください．  

`money`に10000，`rate`に120を代入したときの出力例:  
```
10000円は83ドル33セント
```

`money`に15000，`rate`に125を代入したときの出力例：
```
15000円は，120ドル0セント
```
